## Community Property Smart Card Management

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/2208f3843d5042b7ab7e0066e0a1225d/ghb20250917) 

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript` etc.
- Database: `MySQL`

### 2. Related Software (Any one of the listed software can run)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Function Description
The system is divided into (roles): Regular User, Administrator.

**Regular User Functions:**
1. Login
2. Register
3. Change Password
4. Personal Information
5. Parking Registration Management
6. Fee Management
7. Message Board Management
8. Announcement Information Management
9. Owner Service Management
10. Owner Evaluation Management


**Administrator Functions:**
1. Login
2. Personal Center (Change Password, Personal Information)
3. Administrator Management
4. Parking Registration Management
5. Vehicle Status Management
6. Fee Type Management
7. Announcement Type Management
8. Service Type Management
9. Fee Type Management
10. Message Board Management
11. Announcement Information Management
12. Owner Service Management
13. Service Evaluation Management
14. Owner Management
